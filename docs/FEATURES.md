# Features

- Wallet-only login with signed messages and JWT sessions.
- Basket creation with composition, thesis, fees, and optional chain registration.
- Marketplace search, category filters, rating filters, sorting, trending, recommended, and top performers.
- Basket detail pages with composition, charts, metrics, rebalance logs, reviews, and subscription flow.
- Dashboard and portfolio views for created and subscribed baskets.
- Manual rebalancing endpoint and creator-only UI.
- Monthly rebalance scheduler behind `ENABLE_CRON=true`.
- Copy/fork basket flow through backend API.
- Reviews, ratings, advanced analytics, performance history, and fee accounting.
- PWA-ready responsive frontend.
- Smart contracts for registry, vault, rebalancing events, and fee distribution.
